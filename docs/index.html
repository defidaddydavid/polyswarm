How it works

🧠

12 Distinct Agents

Each agent has a unique expertise, information focus, and documented bias — creating genuine disagreement, not echo-chamber consensus.

⚔️

Multi-Round Debate

Agents see each other's reasoning and update beliefs across multiple rounds. Weak arguments collapse. Strong ones survive.

📡

15+ Live Data Feeds

Every forecast is grounded in real-time data — funding rates, options flow, on-chain metrics, social sentiment, DeFi TVL — fetched in parallel.

📊

Self-Calibrating

Brier scores track each agent's accuracy over time. Better-calibrated agents automatically receive more weight. The swarm improves.

🎭

Scenario Simulation

Not just "will X happen" — simulate what happens IF X happens. Crowd reactions, sentiment shifts, price impact, second-order effects.

🔌

Any LLM Provider

Claude, GPT-4o, Llama, Mistral, Ollama — swap providers with one env var. Run fully local and private if you want.

Get Started in 60 Seconds

terminal 
# Clone & install
$ git clone https://github.com/defidaddydavid/polyswarm.git
$ cd polyswarm && pip install -r requirements.txt

# Add your API key
$ cp .env.example .env

# Run your first forecast
$ python main.py forecast "Will BTC hit $100k before July?" --odds 0.45

# Or simulate a scenario
$ python main.py scenario "SEC bans crypto staking"

# Or start the API server
$ python main.py serve
→ http://localhost:8000/docs
